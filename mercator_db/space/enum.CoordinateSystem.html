pub enum CoordinateSystem {
    Universe {
        origin: Position,
    },
    AffineSystem {
        origin: Position,
        axes: Vec<Axis>,
    },
}

Kinds of space coordinate systems, or bases

Variants

Universe

Absolute base, which allows to generate transformation between spaces by anchoring them relative to each other.

Fields of Universe

origin: Position

A position which contains zeroes for all its coordinates, but has a coordinate per dimensions of the highest dimensions space referenced.

AffineSystem

Base which needs only an affine transformation to map into the Universe.

Fields of AffineSystem

origin: Position

Coordinates in Universe, expressed in f64, or decoded, and in the Universe number of dimensions.

axes: Vec<Axis>

The definition of the coordinate system, through its axes.
